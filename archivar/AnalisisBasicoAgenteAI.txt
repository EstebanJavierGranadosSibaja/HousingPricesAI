### Sistema para la Predicción de Precio de Viviendas

##¿Cuál es el agente?
#El agente se encarga de valorar posibles precios para viviendas según parametros que el usuario le da.

##¿Cuál es el entorno?
#La base de datos historica, la interfaz que utiliza el usuario y el mercado inmoviliario.

##¿Cuál es la función de agente?
#EL agente toma las caracteristicas que se le dan para lograr hacer procesos como regresiones lineales o caluculos estadisticos que le permitan definiar cual es el valor de una vivienda con dichas caracteristicas, lo que se puede considerar un mapeo.

##Cuál es la medida de desempeño?
#El valor que devuelve sea coherente con opciones similares en caracteristicas y que el valor final sea lo más cercano al valor dado por el agente.

##¿Qué tipo de entorno es?
#Totalmente Observable

#Determinista (el modelo devuelve siempre el mismo valor para los mismos datos).

#Episódico (cada predicción es independiente).

#Estático.

#Continuo (por la variable precio).
